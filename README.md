# Web Monitor Automation

A Python automation tool designed to monitor websites, detect changes, and send notifications automatically.

## Overview

This project demonstrates how Python can be used to automate repetitive monitoring tasks by extracting information from websites and notifying users when relevant changes occur.

## Features

- Website monitoring automation
- Data extraction from web pages
- Change detection system
- Notification alerts
- Configurable monitoring settings

## Technologies

- Python
- Playwright
- BeautifulSoup
- Requests
- Git

## Use Cases

Examples of real-world applications:

- Price monitoring
- Availability tracking
- Website change alerts
- Data collection automation
- Business process automation
